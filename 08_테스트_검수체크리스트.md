# 테스트 및 검수 체크리스트

## 1. 문서 목적
본 문서는 AX/DX 20시간 과정 웹 서비스의 개발 완료 전 테스트 기준을 정의한다.

## 2. 기본 화면 테스트
| TC ID | 테스트 항목 | 기대 결과 |
|---|---|---|
| TC-001 | 메인 페이지 접속 | Top Menu, Left Menu, Body가 표시된다. |
| TC-002 | 20개 강의 표시 | 전체 20개 강의가 확인된다. |
| TC-003 | 단원 선택 | 해당 단원의 강의 목록이 표시된다. |
| TC-004 | 강의 선택 | Left Menu가 해당 강의 세부목차로 변경된다. |
| TC-005 | 세부목차 선택 | Body 내용이 변경된다. |
| TC-006 | 선택 상태 강조 | 현재 선택 강의와 세부목차가 강조된다. |
| TC-007 | 산출물 표시 | Body 하단에 산출물이 표시된다. |
