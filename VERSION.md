# VIBELECTURE 버전 관리

## 현재 기준 버전

- 버전: `v1.0.0`
- 확정일: 2026-06-20
- 상태: 최종 기준본
- 운영 주소: `http://127.0.0.1:3001`

## 주요 반영 사항

- 20개 교시 강의 콘텐츠 내비게이터 구성
- 단원명 `AX/DX 기획` 적용
- `AX/DX 후보`, `AX/DX 과제`, `AX/DX 서비스` 용어 통일
- 강의 상세 콘텐츠, 실습, 체크리스트, 산출물 표시
- 접근 코드 기반 강의실 입장 기능
- 반응형 데스크톱·모바일 화면

## 관리 원칙

1. 소스 또는 문서 수정 전 현재 Git 상태를 확인한다.
2. 기능 단위로 수정하고 데이터 검증, 타입 검사, 운영 빌드를 수행한다.
3. 검증이 완료된 변경만 Git 커밋으로 저장한다.
4. 배포 기준본은 의미 있는 버전 태그로 표시한다.
5. `.env.local`, 빌드 결과물, 실행 로그는 Git에 저장하지 않는다.

## 검증 명령

```bash
npm.cmd run validate:data
npm.cmd run typecheck
npm.cmd run build
```

## 운영 서버 재기동

```bash
npm.cmd run start -- --hostname 127.0.0.1 --port 3001
```
