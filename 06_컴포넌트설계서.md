# 컴포넌트 설계서

## 1. 문서 목적
본 문서는 React 또는 Next.js 기반으로 구현할 주요 컴포넌트를 정의한다.

## 2. 컴포넌트 구조
```text
App
├── TopNavigation
├── UnitTabs
├── LessonSelector
├── LessonLayout
│   ├── SideNavigation
│   └── ContentBody
│       ├── LessonHeader
│       ├── SectionContent
│       ├── ActivityBox
│       ├── OutputCard
│       └── PrevNextButtons
└── Footer
```

## 3. TopNavigation
서비스명과 주요 메뉴를 표시한다.

```ts
type TopNavigationProps = {
  title: string;
};
```

## 4. UnitTabs
단원별로 강의를 분류하여 보여준다.

```ts
type UnitTabsProps = {
  units: string[];
  selectedUnit: string;
  onSelectUnit: (unit: string) => void;
};
```

단원: 입문, AX/DX 기획, 개발문서 설계, AI·API·리스크, 운영·출시

## 5. LessonSelector
선택된 단원에 속한 강의 목록을 표시한다.

```ts
type LessonSelectorProps = {
  lessons: Lesson[];
  selectedLessonId: string;
  onSelectLesson: (lessonId: string) => void;
};
```

## 6. SideNavigation
선택된 강의의 세부목차를 표시한다.

```ts
type SideNavigationProps = {
  lesson: Lesson;
  selectedSectionId: string;
  onSelectSection: (sectionId: string) => void;
};
```

## 7. ContentBody
선택된 세부목차의 상세내용을 표시한다.

```ts
type ContentBodyProps = {
  lesson: Lesson;
  section: LessonSection;
};
```

## 8. OutputCard
차시별 산출물을 카드로 보여준다.

```ts
type OutputCardProps = {
  output: string;
};
```

## 9. EmptyState
데이터가 없을 때 안내 메시지를 표시한다.

```ts
type EmptyStateProps = {
  message: string;
};
```

## 10. PrevNextButtons
이전/다음 세부목차 또는 강의로 이동한다.

```ts
type PrevNextButtonsProps = {
  onPrev?: () => void;
  onNext?: () => void;
  prevLabel?: string;
  nextLabel?: string;
};
```

## 11. 상태 관리
MVP에서는 React useState로 충분하다.

```ts
const [selectedUnit, setSelectedUnit] = useState("입문");
const [selectedLessonId, setSelectedLessonId] = useState("L001");
const [selectedSectionId, setSelectedSectionId] = useState("S001-01");
```

## 12. 컴포넌트 완료 기준
- 각 컴포넌트는 독립적으로 재사용 가능해야 한다.
- 데이터는 props로 전달한다.
- 컴포넌트 내부에 강의 데이터를 하드코딩하지 않는다.
- 선택 상태는 상위 컴포넌트에서 관리한다.
