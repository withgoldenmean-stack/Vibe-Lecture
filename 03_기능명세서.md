# 기능명세서

## 1. 문서 목적
본 문서는 AX/DX 바이브코딩 20시간 과정 웹 서비스의 기능 요구사항을 정의한다.

## 2. 기능 우선순위 기준
| 우선순위 | 의미 |
|---|---|
| P0 | MVP 필수 기능 |
| P1 | 2차 개발 권장 기능 |
| P2 | 향후 확장 기능 |

## 3. MVP 기능 목록
| 기능 ID | 기능명 | 설명 | 우선순위 |
|---|---|---|---|
| F-001 | 강의 데이터 표시 | 20개 강의 데이터를 화면에 표시한다. | P0 |
| F-002 | Top Menu | 상단에서 강의 또는 단원을 선택한다. | P0 |
| F-003 | Left Menu | 선택된 강의의 세부목차를 표시한다. | P0 |
| F-004 | Body 출력 | 선택된 세부목차의 상세내용을 표시한다. | P0 |
| F-005 | 선택 상태 강조 | 현재 강의와 세부목차를 시각적으로 강조한다. | P0 |
| F-006 | 반응형 레이아웃 | PC, 태블릿, 모바일에서 사용 가능하게 한다. | P0 |
| F-007 | 산출물 표시 | 각 교시의 산출물을 Body 영역에 표시한다. | P0 |
| F-008 | 빈 상태 안내 | 데이터가 없을 때 안내문을 표시한다. | P0 |
| F-009 | 이전/다음 이동 | 앞뒤 강의 또는 세부목차로 이동한다. | P1 |
| F-010 | 키워드 검색 | 강의명, 세부목차, 본문 검색을 제공한다. | P1 |
| F-011 | 진도 체크 | 사용자가 본 세부목차를 체크한다. | P1 |
| F-012 | 관리자 CMS | 강의 데이터를 웹에서 수정한다. | P2 |

## 4. Top Menu 기능
- 서비스명을 표시한다.
- 단원 또는 강의를 선택할 수 있다.
- 강의 선택 시 Left Menu와 Body가 갱신된다.
- 20개 강의를 모두 가로로 나열하지 말고 단원 탭 또는 드롭다운 구조를 권장한다.

## 5. Left Menu 기능
- 선택된 강의의 세부목차를 표시한다.
- 현재 선택된 세부목차를 강조한다.
- 강의 산출물을 요약해서 보여준다.
- 세부목차 클릭 시 Body 영역을 변경한다.

## 6. Body 콘텐츠 출력 기능
표시 항목은 단원명, 교시 번호, 강의명, 세부목차 제목, 상세교육내용, 실습 과제, 산출물, 이전/다음 버튼이다.

## 7. 반응형 기능
| 화면 | 구조 |
|---|---|
| PC | Top Menu + 좌측 Left Menu + 우측 Body |
| 태블릿 | Top Menu + 접힘형 Left Menu + Body |
| 모바일 | Top Menu + 강의 선택 드롭다운 + 세부목차 드롭다운 + Body |

## 8. 빈 상태 및 오류 상태
| 상황 | 메시지 |
|---|---|
| 강의 없음 | 등록된 강의가 없습니다. |
| 세부목차 없음 | 등록된 세부목차가 없습니다. |
| 본문 없음 | 등록된 상세교육내용이 없습니다. |
| 검색 결과 없음 | 검색 결과가 없습니다. |
| 데이터 로딩 실패 | 강의 데이터를 불러오지 못했습니다. |

## 9. 검색 기능, 2차 개발
검색 대상은 강의명, 단원명, 세부목차 제목, 상세내용, 산출물이다. 검색 결과 클릭 시 해당 강의와 세부목차로 이동한다.

## 10. 완료 기준
- 20개 강의가 모두 데이터로 등록되어 있다.
- Top Menu 클릭 시 강의가 변경된다.
- Left Menu 클릭 시 Body 내용이 변경된다.
- 현재 선택 상태가 명확히 보인다.
- 모바일 화면에서 콘텐츠를 읽을 수 있다.
- 데이터 누락 시 안내 메시지가 표시된다.
